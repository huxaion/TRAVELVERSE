# Database Schema Overview

Prisma schema includes all requested entities:
User, Profile, Country, City, Destination, Hotel, Restaurant, Activity, Tour, Event, Flight, Booking, Itinerary, ItineraryDay, ItineraryItem, SavedItem, Post, Comment, Like, Follow, Review, Report, Notification, TravelGuide, Checklist, BudgetPlan, GroupTrip, GroupTripMember, Vote, ScamAlert, SafetyTip, CulturalTip, LocalPhrase.

Database provider: PostgreSQL.
